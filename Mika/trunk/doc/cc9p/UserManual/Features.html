Features

The CC9P Development Kit for Java incorporates the Mi|k|a VM and class libraries from /k/ Embedded Java Solutions.

  • VM and class libraries based on Wonka(R), with support for the following APIs:

    • java.io, java.lang, java.lang.ref, java.lang.reflect, java.math
    • java.net, java.rmi, java.sql, java.text
    • java.util, java.util.jar, java.util.zip java.security.interfaces, java.security.spec
    • javax.comm, javax.crypto, javax.microedition.io
    • partial support: java.beans, java.rmi.activation, java.rmi.dgc

    If the AWT option has been purchased then the following packages are also available:

    • java.applet, java.awt, java.axt.dnd, java.awt.image
  • JNI support, with the exception of the Activation API.

  • Java applications can be developed using any Java development environment, including vi.

  • No root acess required for development or testing.

  • Entire Java system can be run either NFS-mounted or from flash.
