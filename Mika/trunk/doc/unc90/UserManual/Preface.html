Preface

The UNC90 Development Kit for Java provides a complete solution for executing Java2TM programs on the UNC90 development systems supplied by FS Forth-Systeme GmbH. This document explains how to use the Development Kit to develop and test Java applications for the UNC90.

It is assumed that you have access to a UNC90 Development Kit and accompanying documentation, and that you are familiar with the basics of writing and compiling Java programs. You will also need to be able to perform basic Unix/Linux administration tasks such as creating a directory or a symbolic link.
