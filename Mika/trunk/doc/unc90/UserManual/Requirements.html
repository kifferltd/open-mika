Requirements

To use the UNC90 Development Kit for Java, you will need the following:

  • A UNC90 Development Kit from FS Forth Systeme GmbH. To use the AWT option, you will also need an LCD screen with touchscreen overlay.

  • A pc running a reasonably current version of Linux. The pc you are using for the UNC90 development software will do.

  • A Java development system of some kind. As a minimum you may use the jikes compiler from IBM, together with your favourite editor. For convenience a copy of jikes is included on the CD.
