Requirements

To use the Source Code Kit, you will need the following:

  • A UNC20 Development Kit from FS Forth Systeme GmbH.

  • The UNC20 Development Kit for Java.

  • A pc running a reasonably current version of Linux. The pc you are using for the UNC20 development software will do.

  • The following additional tools:

    • Acunia's version of the jam build tool (acunia-jam).

    • Acunia's jamjar tool, a variant of jar for use with jam.

    • A Java compiler such as IBM's jikes or Sun's javac.

The acunia-jam and jamjar tools are included on the Souce Code Kit CD. The jikes compiler is included as part of the UNC20 Development Kit for Java.
