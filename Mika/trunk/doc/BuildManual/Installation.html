Installation

The following description assumes that the UNC20 Development Kit for Java has already been installed, so install it now if you didn't do so already.

Installing the Acunia tools

The acunia-jam and jamjar tools are included on the CD in directory tools/Acunia/bin. To install them simply copy the executable to some directory in your PATH, e.g. an existing directory such as /usr/bin or one you create specially for the purpose. (Note that you will need to become root to install in /usr/bin).

Installing the Mika source tree

The Mika source tree can be installed anywhere you like, e.g. under your home directory or in a shared directory such as /targets/LxNETES/project_unc20. No use is made of symbolic links or other tricky stuff, so any means of copying is good: cp -r, tar, or drag-and-drop in your favourite GUI.

The Mika/ directory contains files COPYRIGHT and COPYRIGHT.Wonka which you should read carefully.

Using a Java compiler other than Jikes

If you decide not to use the Jikes compiler, you will need to overrule the default settings in the top-level Jamrules. In theory you could do this by typing e.g. 

-sJAVAC=javac
each time you invoke acunia-jam, but in practice it is probably easier to open Mika-source/Jamrules in a text editor and change the line 
JAVAC default = jikes ;
to select your preferred Java compiler.

Note that the Mika class libraries represent a large and complex project, and some compilers may produce incorrect bytecode for some components. Therefore we recommend sticking to the version of Jikes supplied as part of the UNC20 Development Kit for Java.
