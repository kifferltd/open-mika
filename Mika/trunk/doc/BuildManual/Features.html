Features

The UNC20 Development Kit for Java incorporates the Mi|k|aTM VM and class libraries from /k/ Embedded Java Solutions.

  • VM and class libraries based on Wonka(R), with support for Java2 APIs up to J2ME CDC/PP.

  • Java applications can be developed using any Java development environment, including vi.

  • No root acess required for development or testing.

  • Entire Java system can be run either NFS-mounted or from flash.

Using the Source Code Kit it is possible to create a version of Mi|k|a tailored to your needs, for instance by omitting unnecessary packages or by providing access to native functionality using the Java Native Interface (JNI). The Source Code Kit is also invaluable in helping to track down problems which arise in using Mi|k|a.
